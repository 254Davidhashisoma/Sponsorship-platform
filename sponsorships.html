Google Africa Developer Scholarship

Registration is now open for the Google Africa Developer Scholarship program supported by Grow with Google, Andela, and Pluralsight. Software engineers (and aspiring software engineers) looking to build competencies in Android, Google Cloud, and Mobile Web (MWS) are encouraged to apply to the program. The aim of this program is to continuously engage with aspiring and existing software engineers on these tracks a nd equip them with the requisite skills to get certified.
